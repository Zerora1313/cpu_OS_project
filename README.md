# 🧠 CPU Scheduling Simulator

A modular and extensible CPU scheduling simulator built using Object-Oriented Design in C++.

---

## ✨ Features
- ⚙️ Supports multiple scheduling algorithms:
  - Round Robin
  - Preemptive SJF
  - Priority Scheduling
- 🧩 Modular and extensible OOP architecture
- 📊 Performance comparison between algorithms

---

## 🛠 Tech Stack
- C++
- Object-Oriented Programming (OOP)

---

## 🧠 Key Concepts
- OS Scheduling Algorithms
- Abstraction & Polymorphism
- Process Lifecycle Simulation

---

## 🏗 Architecture
- Abstract Base Class: Scheduler
- Derived classes for each scheduling algorithm
- Centralized Process model

---

## 🚀 Getting Started

git clone <repo-link>  
cd CPUScheduler  
g++ main.cpp -o scheduler  
./scheduler
